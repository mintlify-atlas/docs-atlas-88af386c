EmbyTok connects to your Emby server and lets you browse specific libraries or all your media at once. Customize which libraries appear and mark your favorites for quick access.

## Selecting a Library

Navigate to different media libraries to focus your browsing:

1. Tap the **Menu icon** (three lines) in the top left corner
2. Your available libraries appear in the list
3. Tap any library to filter the feed to that library only
4. Select **"All Media"** to browse videos from all libraries

The currently selected library name appears as a subtle badge below the top navigation bar.

<Info>
  When you switch libraries, the feed automatically reloads and resets to the beginning with your selected feed mode (Latest/Random/Favorites).
</Info>

## Hiding Libraries

Keep your library list clean by hiding libraries you don't need:

1. Open the menu and tap **"Settings"** at the bottom
2. Scroll to the **"Library Management"** section
3. Each library has an eye icon on the right:
   - **Eye open** (blue): Library is visible
   - **Eye closed** (gray): Library is hidden
4. Tap the eye icon to toggle visibility

Hidden libraries won't appear in the main library selection list but can be re-enabled anytime from settings.

<Note>
  Hiding a library doesn't affect your Emby server—it only controls what you see in EmbyTok.
</Note>

## Favorites System

Mark videos you love and access them quickly through the Favorites feed mode.

### Adding Favorites

While viewing any video:

1. Look for the **heart icon** on the right side of the screen
2. Tap the heart to mark the video as a favorite
3. The heart fills with red to confirm

### Viewing Favorites

1. Switch to **Favorites** feed mode (from the top bar or menu)
2. Only your favorited videos appear
3. Works across all libraries or filtered to a specific library

<Accordion title="How are favorites stored?">
  EmbyTok syncs favorites directly with your Emby server using the built-in favorites system. This means:
  
  - Favorites persist across devices
  - Favorites appear in other Emby clients
  - You can favorite videos in Emby web/apps and see them in EmbyTok
  - Favorites are tied to your user account
  
  For local folder mode and folder server mode, favorites are stored locally in browser storage.
</Accordion>

## Library Information

EmbyTok automatically detects all video libraries from your Emby server, including:

- Movies
- TV Shows (individual episodes)
- Home Videos
- Any custom video collections

<Tip>
  If a library doesn't appear, make sure it's configured as a video library in your Emby server settings and that your user account has access to it.
</Tip>

## Per-Library Feed Modes

Feed modes work independently with library selection:

- **Latest + Specific Library:** Shows newest videos from that library only
- **Random + All Media:** Shuffles videos from every library
- **Favorites + Specific Library:** Shows only favorited videos from that library

This flexibility lets you browse in the way that makes sense for your content.
